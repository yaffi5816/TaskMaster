# הגדרת מערכת התזכורות במייל

## שלב 1: הגדרת Gmail App Password

1. היכנס ל-Google Account שלך
2. עבור ל-Security > 2-Step Verification
3. גלול למטה ל-"App passwords"
4. צור App Password חדש עבור "Mail"
5. העתק את הסיסמה שנוצרה (16 תווים)

## שלב 2: עדכון קובץ .env

פתח את הקובץ `.env` ועדכן:

```
GMAIL_USER=your_email@gmail.com
GMAIL_PASSWORD=your_16_char_app_password
```

## שלב 3: הפעלת השרת

```bash
python -m uvicorn main:app --reload
```
