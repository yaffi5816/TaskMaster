# 🎨 ממשק משתמש - מערכת ניהול משימות

## איך להריץ?

### שלב 1: הרץ את השרת
```bash
cd ..
uvicorn main:app --reload
```

### שלב 2: פתח את הממשק
פשוט פתח את הקובץ `index.html` בדפדפן!

או השתמש ב-Live Server אם יש לך ב-VS Code.
